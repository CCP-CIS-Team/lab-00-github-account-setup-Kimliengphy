## What You Will Do

1. Accept this assignment (you already did — nice work!)
2. Edit the `profile.md` file with your information
3. Commit and push your changes
4. Done! ✅

---

## How to Edit on GitHub (No Git Install Needed)

1. Click on `profile.md` in this repo
2. Click the **pencil icon** (Edit this file) in the top right corner
3. Fill in your information — replace the placeholder text
4. Scroll down to **Commit changes**
5. Leave the default message or write your own, then click **Commit changes**

That is it! Your profile is submitted.

---

## Optional: Edit Locally with Git

If you have Git installed:
```bash
git clone <your-repo-url>
cd <your-repo-folder>
# Edit profile.md in any text editor
git add profile.md
git commit -m "Add my profile"
git push
```
